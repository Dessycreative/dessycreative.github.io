Forecasting Model::01

Context-Aware Sales Forecasting Model

Built entirely in Power BI using DAX — no Python, no external tools. Forecasts future sales using a rolling 6-week actuals window with full context awareness. Bank holidays, seasonal events, and special occasions are baked into the logic so the model never blindly projects off a distorted trading week. Weighted recency ensures recent performance carries more influence. Fully interactive via slicers — business users engage directly, no analyst intervention required.

Finance Model::02

Three-Statement Financial Model — PnL, Balance Sheet & Cash Flow

Full three-statement model in Power BI with Excel as the data source layer. PnL tracks revenue, EBITDA and cost categories through a Chart of Accounts structure. Cash Flow generates AI narrative summaries as DAX Card visuals — auto-updating on every filter. Balance sheet drill-through exposes all subcodes behind every consolidated line item.
